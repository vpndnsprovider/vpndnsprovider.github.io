Install WireGuard on Android

  1. Open Google Play Store.
  2. Search for WireGuard and install the app.

Import Configuration

  1. Open the WireGuard app.
  2. Tap the + icon at the bottom right.
  3. Choose "Import from file or archive".
  4. Select the .conf file you downloaded.
  5. Toggle the switch to activate the VPN.
