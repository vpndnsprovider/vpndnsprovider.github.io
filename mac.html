Install WireGuard on macOS

  1. Open the Mac App Store.
  2. Search for WireGuard and install it.

Import Configuration

  1. Launch the WireGuard app.
  2. Click "Add Tunnel" > "Add Empty Tunnel" or "Import Tunnel(s) from File...".
  3. Choose your downloaded .conf file.
  4. Click Activate to start the VPN.
